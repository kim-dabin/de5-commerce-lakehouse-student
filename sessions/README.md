# 차시별 실습 폴더

각 폴더는 해당 차시 수강생 배포용 자료입니다.

```text
01-architecture-blueprint
02-local-docker-stack
03-kafka-producer
04-flink-paimon-bronze
05-spark-iceberg-batch-bi
06-airflow-orchestration
07-realtime-olap-bi
08-final-portfolio
```

수업 전에는 현재 폴더 어디에서든 `git pull`을 실행해 최신 자료를 받은 뒤, 해당 차시 폴더의 `DOWNLOAD_GUIDE.md`를 확인합니다.
